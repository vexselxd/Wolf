# 🐺 Wolf

> Where it started.

Wolf is the original version of what later became **Wolf 2.0**. Built as a final project for the **Kodland Python Basics** certification — the first time I made a complete, playable game from scratch.

Simple in every way. That was the point.

## What it is

A basic survival game built with PGZero. Enemies spawn, you shoot, you survive. No boss, no upgrades, no complex systems — just the core loop working.

It was enough to finish the certification. It was also enough to want to build something better.

---

## Running the Game

```bash
pip install pgzero
pgzrun main.py
```
